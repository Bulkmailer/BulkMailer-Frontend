### Running locally

1. Clone the repository (for contribution fork and clone)

```sh
git clone https://github.com/Bulkmailer/BulkMailer-Frontend.git
```

2. Install packages (Use npm)

```sh
cd BulkMailer-Frontend
npm install
```

3. Local development

```sh
npm run start
```

## Tech Stack

- ReactJs
- Redux
- Redux Thunk
- Ant Design
- JavaSCript
